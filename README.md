# Heritage(Namma) Shasane

Heritage Shasane is a specialized Android application designed to document, preserve, and explore historical stone inscriptions (Shasane). The app bridges the gap between ancient history and modern technology by providing interactive maps, AI-driven insights, and educational tools.

## 🚀 Features

*   **Interactive Inscription Map:** Locate historical inscriptions geographically using Google Maps integration.
*   **AI Decode & Digital Repository:** Detailed information on inscriptions including Dynasty, Century, and historical context (e.g., Halmidi Inscription).
*   **Heritage Quiz:** Test your knowledge about regional history and inscriptions through an interactive quiz module.
*   **Photo Tagging:** Capture and contribute to the digital documentation of heritage sites.
*   **Offline Access:** Powered by Room Database for efficient local data management.

## 🛠 Tech Stack

*   **Language:** Kotlin
*   **Database:** Room Persistence Library
*   **Maps:** Google Maps SDK for Android
*   **Architecture:** MVVM (Model-View-ViewModel)
*   **Jetpack Components:** Navigation Component, ViewBinding, Flow, ViewModel

## 📋 Prerequisites

*   Android Studio Ladybug or later
*   JDK 17+
*   Google Maps API Key (to be added in `local.properties` or `AndroidManifest.xml`)

## ⚙️ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/neeraj18k/major_project.git
   ```
2. Open the project in Android Studio.
3. Sync the project with Gradle files.
4. Add your `MAPS_API_KEY` to `local.properties`.
5. Run the app on an emulator or physical device.
